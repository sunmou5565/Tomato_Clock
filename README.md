# 番茄时钟
基于Electron框架开发的简洁番茄时钟程序

---
## 目录
 - [使用库](#库)
 - [启动源码](#启动源码)
 - [编译](#编译)
 - [介绍](#介绍)
 - [许可证](#许可证)
---
## 库
本项目使用Electron框架作为主体开发框架，可以使用npm或cnpm安装他
```shell
npm install electron --save-dev
cnpm install electron --save-dev
```
## 启动源码
进入项目目录，执行```npm start```即可启动源码

## 编译
进入目录，执行命令
```shell 
npm run dist
```

## 介绍
本项目基于番茄时间工作法设计，工作25分钟休息5分钟，四轮番茄时长时间休息一次（25分钟）
 - 参考：https://en.wikipedia.org/wiki/Pomodoro_Technique
